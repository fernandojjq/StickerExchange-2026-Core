## Diagnóstico Inicial (Preguntas Clave)

1. **¿La web está indexada?**
   - Ejecutar: `site:swap26.pages.dev` en Google
   - Si no aparece: Verificar Google Search Console

2. **¿Tiene sitemap.xml?**
   - Verificar: swap26.pages.dev/sitemap.xml
   - Si no existe: Crear uno

3. **¿Tiene robots.txt?**
   - Verificar: swap26.pages.dev/robots.txt
   - Si no existe: Crear uno permissive

4. **¿Meta tags básicos?**
   - Title, description, canonical URL
   - Open Graph tags para social sharing

5. **¿Contenido suficiente?**
   - Google necesita contenido para indexar
   - Mínimo 3-5 páginas con contenido único
